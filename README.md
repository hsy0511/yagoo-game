랜덤으로 3개에 숫자가 나옵니다.
3개에 숫자중에 숫자는 맞고 자리는 다를 경우는 ball이 나오고 숫자와 자리가 모두 맞으면 strike가 나옵니다.
3개에 숫자와 자리가 다 맞아서 3strike가 나오면 게임이 종료 됩니다.
